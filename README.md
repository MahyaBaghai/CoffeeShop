# ☕ Coffee Shop E-Commerce | Frontend Project

****A responsive online coffee store built with ***React, JavaScript, & Tailwind CSS***.****  

## 🚀 Features

- 📱 ***Fully Responsive*** 
- 🌙 ***Dark/Light Theme***  
- 🌍 ***Multilingual (EN/DE)*** 


### 🛠️ Tech Stack

- React 19 (Context API/Hooks)
- Tailwind CSS 4.1
- React Router 6 (Routing foundation)
- Vite (Build tool)
- Webpack 5
